**TechGate** é uma Landing Page de alta conversão projetada especificamente para empresas de manutenção técnica e automação. O projeto foca em **UX (User Experience)**, **Performance** e **Gatilhos de Venda** para transformar visitantes em clientes reais via WhatsApp.

---

## 🚀 Diferenciais do Projeto

### 1. Engenharia Visual e Performance
- **Gate Animation:** Simulação de um portão automático com scanner de segurança feito 100% em CSS (Keyframes), garantindo leveza e carregamento instantâneo.
- **Glassmorphism:** Interface moderna com efeitos de desfoque e transparência na navegação.
- **Mobile First:** Totalmente responsivo para smartphones, onde ocorre a maioria das solicitações de serviço urgente.

### 2. Estratégias de Conversão (Marketing Tech)
- **Prova Social Dinâmica:** Sistema em JavaScript que exibe notificações de novos clientes em tempo real, gerando autoridade.
- **Gatilhos de Escassez:** Barra de urgência superior e cards com vagas limitadas para bônus de lançamento.
- **Smart WhatsApp Form:** Formulário que processa os dados do cliente e já abre o chat com a mensagem estruturada, reduzindo a fricção no atendimento.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5 Semântico:** Estrutura focada em SEO para melhor ranqueamento no Google.
- **CSS3 Moderno:** Uso de Variáveis (CSS Variables), Flexbox e Grid Layout.
- **Vanilla JavaScript:** Lógica pura para manipulação de DOM e notificações.
- **ScrollReveal.js:** Biblioteca para animações de entrada fluidas durante a rolagem.
